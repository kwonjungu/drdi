# 디싹 캠프 서류 자동화 (GitHub Pages · 서버리스)

캠프명단(xlsx) 한 번 업로드로 디지털새싹 캠프 제출 서류를 자동 작성·다운로드합니다.

- **완전 클라이언트(서버리스)** — 모든 처리는 브라우저 안에서만. 서버·DB 없음.
- **개인정보 저장/전송 없음** — 명단·서류 데이터가 외부로 나가지 않습니다.
- 별도 설치/로그인 없이 누구나 사용.

## 생성 서류
- 학생: 수업신청학생등록양식(xlsx, 익명화)
- 🔵 주강사: 교구관리대장 · 주강사료지급신청서 · 수령대장
- 🟢 보조강사: 결과보고서 · 강사료지급신청서 · 다문화 확인서 · 수령대장
- 🟠 안전관리자: 안전관리 이수증(빈양식) · 안전관리 서약서 · 안전업무일지 · 안전체크리스트 · 운영 사례집 · 안전 지급신청서 · 안전 근로계약서

## 사용
1. 표준 캠프명단 양식 내려받아 작성 → 업로드
2. 차시/일정/담당자(주·보조·안전) 설정
3. **변환하기** → 역할별 버튼에서 서류 다운로드

## 호스팅
정적 파일만으로 동작 → GitHub Pages에서 바로 서빙.
(저장·공유 링크가 필요한 지인용 버전은 Vercel + Firebase 별도 운영)
